
# What is this app ?
- A very basic UI to get a text input
- The model is loaded locally
- The prediction is done locally

## How to install it
- Move a Terminal inside the streamit directory

```
cd detect_ai_content/streamlit
```

- Install the dependencies

```
make install_on_dev
```
